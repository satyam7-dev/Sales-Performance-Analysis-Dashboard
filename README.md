# 📊 Sales Performance Analysis Dashboard

A comprehensive data analytics project demonstrating the full data lifecycle: **Extraction (SQL)**, **Cleaning (Excel)**, and **Visualization (Tableau)**. This dashboard helps stakeholders analyze sales trends, profitability, and regional performance to drive strategic decisions.

## 🛠 Tech Stack
- **SQL**: MySQL/PostgreSQL (Data Modeling & Querying)
- **Excel**: Data Cleaning, Validation, Pivot Tables
- **Tableau**: Interactive Dashboards, KPIs, Calculated Fields
- **Git/GitHub**: Version Control & Portfolio Hosting
